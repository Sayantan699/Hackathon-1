C++ Overview


What is C++?

  • C++ was developed by Bjarne Stroustrup, as an extension to the C language.
  • Despite being an 80s creation, C++ has been a popular programming language throughout these years.
  • C++ is a cross-platform language that can be used to create high-performance applications and software systems.
  • C++ is very close to the hardware making it comparatively easy for programmers to give the instructions directly to the system without any intermediary giving programmers a high level of control over system resources and memory.

Why should we learn C++/ Features of C++?

  • C++ is one of the world's most popular programming languages.
  • In today's operating systems, GUIs, and embedded systems, C++ is widely used.
  • With C++, you can develop applications or heavy games that can run on different platforms.
  • As C++ is close to other programming languages such as C# and Java, which makes it easy for programmers to switch to C++ or vice versa while it is actually very easy to learn.

How is it different from C?

  1. The syntax of C++ is almost identical to that of C, as C++ was developed as an extension of C
  2. n contrast to C, C++ supports classes and objects, while C does not.

Function Declaration

A C++ program is made up of different tokens combined. These tokens include:

  • Keywords
  • Identifiers
  • Constants
  • String Literal
  • Symbols & Operators

  1. Keywords

    2. Keywords are reserved words that can not be used elsewhere in the program for naming a variable or a function. They have a specific function or task and they are solely used for that. Their functionalities are pre-defined.

  2. Identifiers

    3. Identifiers are names given to variables or functions to differentiate them from one another. Their definitions are solely based on our choice but there are a few rules that we have to follow while naming identifiers. One such rule says that the name can not contain special symbols such as @, -, *, <, etc.

  3. Constants

    4. Constants are very similar to a variable and they can also be of any data type. The only difference between a constant and a variable is that a constantâ€™s value never changes. We will see constants in more detail in the upcoming tutorials.

  4. String Literal

    5. String literals or string constants are a sequence of characters enclosed in double quotation marks. Escape sequences are also string literals.

  5. Symbois and Operators

    6. Symbols are special characters reserved to perform certain actions. Using them lets the compiler know what specific tasks should be performed on the given data. Several examples of symbols are arithmetical operators such as +, *, or bitwise operators such as ^, &.
